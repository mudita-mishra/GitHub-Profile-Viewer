# GitHub Profile Viewer

A web app that fetches GitHub user data using the GitHub API and displays profile details like followers and repositories. It uses simple logic to classify users into levels, personality types, and experience, with visual progress bars.

---

## Features

- Fetch GitHub user data using API  
- Handle invalid usernames  
- Display profile information dynamically  
- Classify users based on:
  - Followers (Level)
  - Followers vs Following (Personality)
  - Account age (Experience)
- Visual progress bars  
- Loading indicator  
- Enter key support  

---

## Built With

- HTML  
- CSS  
- JavaScript

## Getting Started

### Prerequisites
- A web browser (Chrome, Edge, etc.)

### Installation

1. Download or clone the repository  
2. Open the HTML file in your browser  

---

## Usage

1. Enter a GitHub username  
2. Click **Search** or press **Enter**  
3. View the user profile details
